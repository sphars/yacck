YACCK - Yet Another Classless CSS Kit

YACCK is a classless, responsive CSS kit (framework) that focuses on pure HTML semantic elements. It is inspired by many other similar minimal CSS frameworks, such as MVP.css. Meant for use in simple websites and blogs.
